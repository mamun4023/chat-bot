
# CHATBOT

A modern chatbot assistant that will reply your queries. That helps us to make productive.


## Features

- Authentication
- Removing chat history


## Installation

Install All dependecy

```bash
  npm install
```


    
## Run development server

```bash
  npm run dev
```
